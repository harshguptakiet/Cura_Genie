# Contributing

Thank you for contributing to this project!

## Getting Started

### 1. Fork the Repository

Click the **Fork** button on GitHub to create your own copy of the repository.

### 2. Clone Your Fork

```bash
git clone https://github.com/Madhu-T18/Cura_Genie.git
cd LABEL-ANALYZER
```

### 3. Create a New Branch

```bash
git checkout -b docs/update-contributing-guide
```

Use meaningful branch names such as:
- feat/add-feature
- fix/bug-description
- docs/update-readme

---

## Contribution Guidelines

- Keep pull requests small and focused
- Write clear commit messages
- Follow existing project structure and style
- Update documentation when needed

---

## Pull Request Process

Before submitting a pull request:

1. Ensure your changes are properly tested if applicable
2. Provide a clear description of your changes
3. Link related issues if available

---

## Reporting Bugs

When reporting bugs, please include:
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshots or logs if applicable

Thank you for helping improve this project!
